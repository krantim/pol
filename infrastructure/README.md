# Overview

This repository stores all settings and configuration files for the SCB10x infrastructure.

The idea is to store all configuration of our tools here to be able to apply them with ansible.


# Get started

- Install ansible


# How to execute ansible

    ansible-playbook -c local -i "localhost,"  playbook.yml
